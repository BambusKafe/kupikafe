# 7 Грама Студио Аеродром

## За нас

Добредојдовте во **7 Грама Студио Аеродром** - вашиот избор за премиум кафе производи во Македонија!

Ние сме посветени на тоа да ви го донесеме најдоброто кафе од светски познатите брендови. Нашата пасија е да ви обезбедиме искуство што го прави секој ден посебен.

## Производи

### Капсули за кафе
- **Dolce Gusto капсули** - Богати вкусови за секој тип на кафе
- **Nespresso капсули** - Премиум капсули од светски бренд
- **Lavazza A Modo Mio** - Италијанска класа во вашата чаша
- **Esse Cialde pods** - Квалитет и вкус во секоја капсула

### Сирупи за кафе
- Monin сирупи во различни вкусови

## Технички информации

### Инсталација

```bash
# Инсталирајте ги зависностите
npm install

# Стартувајте го серверот
npm start
```

Серверот ќе работи на порта 3000.
